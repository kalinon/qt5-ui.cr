# qt5-ui

Provides a parser for Qt5 Creator `.ui` xml files.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     qt5-ui:
       github: kalinon/qt5-ui.cr
   ```

2. Run `shards install`
